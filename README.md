<div align="center">
TSun Gaming is a powerful and user-friendly Flutter application designed for Free Fire players to manage their guest accounts eassily. It allows users to inject guest credentials, launch the game directly, and access community resources.
</div>

<div align="center">

## üöÄ Features
</div>

- **Guest Account Injection**: Easily create or update guest account credentials (`guest100067.dat`).
- **Smart Game Launcher**: Automatically detects and launches Free Fire or Free Fire Max.
- **Modern UI/UX**: Beautiful dark-themed interface with glassmorphism effects and smooth animations.
- **Image Slider**: Stay updated with the latest news and banners.
- **One-Click Access**: Quick links to tutorials, social media, and community feedback.
- **Data Management**: Option to clear guest data directly from the app.

## üõ†Ô∏è Installation & Setup
</div>

1. **Clone the repository**:
   ```bash
   git clone https://github.com/TSun-FreeFire/TSun-GAMING.git
   cd TSun-GAMING
   ```


2. **Get dependencies**:
   ```bash
   flutter pub get
   ```

3. **Build the APK**:
   - For a universal APK:
     ```bash
     flutter build apk --release
     ```
   - For split APKs (reduced size per architecture):
     ```bash
     flutter build apk --split-per-abi
     ```

The generated APK will be located at:
`build/app/outputs/flutter-apk/app-release.apk`
